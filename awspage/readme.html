はじめにお読みください

2022年9月21日新規

この一連の資料は、
何となく昔から古いホームページを運用している知り合いにインスパイアされて、「昔ながらのホームページ（ここからホムペとします）」を設置してみたくなった。

というスタート地点からの作業記録をまとめています。

そもそも、

  1. サーバ用ハードウェア購入
  2. UNIX系OSインストール
  3. Webなどのサーバ構築
  4. ゴリゴリのHTML直打ちでホムペづくり（エディタは基本vi）
みたいな、かつての情報系オタク学生の必須科目（そうか？）を「昔取った杵柄」でやってみたい。でも、「かつての若者たち」のやり方そのままだと、技術的にそんなに今っぽくなさすぎだし、そこまで自分に無理が効かないし、

ぶっちゃけそこまでコミットするような事はしたくない

みたいな、「ちょっとくたびれたおじさん」にありがちな思いがありまして、「いい機会だから前から気になっていたAWSでサクッと作って、飽きたらサクッと潰そう」くらいの思いで始めてみたわけです。

そんなわけで、なんとなくで始めた割に、途中までは簡単に作れてしまいました。

  1. AWSのアカウントを取得
  2. クラウド上のイントラネットみたいなのを構築
  3. クラウドサーバっぽいやつの構築
  4. クラウド内のWebサーバなどのインストール
  5. その他サーバ上の環境構築
といった作業で得られたノウハウを、ちょいちょいアウトプットしていこうか？と思ったわけであります。

正直なところ、このページはサーバ作業をしていてWebで調べ物をしていると普通に出てくる内容ではあります。それに「昔ながらのホームページ」なわけで、そこまでこのページに本気を出そうとは思ってないつもりです。何か思わしくない部分があっても暖かい目で見ていただきたいです。

そんなわけでよろしくお願いしますm(_ _)m
