HTMLタグ直打ちなアレ

2022年11月17日新規

やっとホムペ書き環境がこなれてきた

先に書いてしまいますが、現状のホームページ運用環境ですが、

  • AWSで自前のWebサーバを構築
  • git pagesで割とお気楽にミラーっぽいサイトを運用
となっております。

※ AWSのサイトは現状非公開な状態で、一部の友人知人にしか告知しておりません。ご了承ください。

HTML直打ち環境は…
  • エディタがVisual Statio Code
  • サーバとの同期はAWSのWebサーバへはrsyncを使った簡単なシェルスクリプト、git PagesへはVS Codeのgit拡張
です。それで、ちょっと前までは、
  • エディタが秀丸エディタ
  • サーバとの同期はAWSのWebサーバへはrsyncを使った簡単なシェルスクリプト、git PagesへはWindows版git
といった感じでした。

別の章に書きましたが、rsyncはcygwinという昔ながらのWindows+Unix的環境構築ツールの配下で動いていて、 実はcygwinを使うより、WSL（Windows Subsystem for Linux）を使って同じ環境がムチャムチャ簡単に作れるので、 Windows11のPro以上の環境をお持ちの方はそっちの方が断然いい感じです。そのうちrsyncの記事を保管する形でお話しするつもりです。

それで、今回の「HTMLを直打ちして、ホームページを作っていくぜ！」というお話をどうしよう？と思っていたのですが、 個別のツールについて、テキトーに語る事でお茶を濁すことにしました。実はこの記事の執筆時に、

  • エビスビール（現在500mlを2本目）
  • 私めが作ったきゅうりと大根のぬか漬け（マジおいしい）
  • 割とこだわりのある食材のお店で買った味噌
を楽しみつつ、退廃を絵に描いたような状況で執筆しておりますので、ちょっと込み入った感じでは書けなさそうな感じです。 なお、ビール1リットルくらいだと、そんなに酔わないです。 スワイショウという中国気功のトレーニングを気が向いた時にチョイチョイやってますと、 そんなに酒に酔わなくなったり、変な感じでショートスリーパーになったりします。 本当に身体に良いかは不明ですが。

Visual Studio Code

とりあえず、真っ先に目に入っているVS Codeについて書いていきます。 私らが学生さんをしていた20年前くらいにはすでに普通に存在していたVisual Studioなわけですが、 当時はVC++用の統合開発環境という感覚で使っていて、VBもそれ以外の言語もあるぜよ、 みたいな感じだったと記憶しております。MacOS用の環境としては、 当時はCode Warriorというカッコよさげな名前の開発環境がありまして、 Windowsとのクロスコンパイルみたいな機能もあったような気がします。 まあ、当然私は大してできる学生さんではなかったので、そんな高度な使い方はしていなかったわけですが。ははは。 なお、こんな話をわざわざするのも、私のターゲット読者は同世代の情報系おじさん達だからです。

で、WebブラウザのIE（Internet Explorer）の頃からか、マイクロソフトは 「割と強力なソフトウェアをガンガンに無償提供して、その市場の支配権を得る」 みたいな戦略を使うようになったと思っていて、Basic開発時代の 「なんで俺のソフトにお金を払ってくれないんや！」 というビル・ゲイツ青年からは想像もつかない動きを見せ始めたわけですね。

まあ、そんなのは割とどうでもいい話ではありまして、テキストエディタというのは 「そこそこ堅牢な入出力」「そこそこ便利な検索置換機能」「そこそこ便利なマクロ機能」があれば、 後はだいたいその人の好み次第なわけで、WindowsというGUIにガッチリハマっているVSが、 Windows環境において隆盛を極めたのは当然といえば当然というわけです。 私もアカデミックパックというのがあったおかげで、学生時代から研究室で愛用していたため、 Visual Studioについては何となく使ってしまう感じです。アレです。 マクドを子供の頃から食べていれば、大人になっても食べ続ける、というアレですね。

まあ～酔っ払っているのか、筆が滑ること滑ること(^ρ^)

で、今は数々の機能拡張があるおかげで、VS CodeはHTMLをガンガンに打ち込む作業にも対応しているわけです。 なので、コーディングが日常の情報系おじさんにしてみれば、自然な選択肢になりえるわけです。まあ、そんな感じです。

すいません。なんか、VS Codeに書いた時点で何となく満足してしまったので、 明日以降酔っ払いながらチビチビ執筆しようと思います。未完です。これも”昔ながらのホムペ”の醍醐味ってことで。
