はじめにお読みください

2022年9月21日新規

はじめに正直に書いてしまいます。色々と生活上の問題から、自分のスキルセットの再構築が必要になりまして。で、とりあえず某Pラーニングを課金してやってみて、SQL入門講座の最後のチャプターの「ちょっとした小技」みたいな回をやってみて「あー、そうそう、こういうの若い頃やったわ～^^;」みたいになって、その後ヘビーすぎる当時の現場の空気を思い出して、素に戻ってやる気をなくしたり。

で、ある同じ大学出身の博士持ちの人にインスパイアされて、「いかにもホームページっぽいホームページ」を書いてみたくなり、Webサーバを立ち上げするために、何となくスキル構築の一つとしてAWSをいじり始めてみたわけです。それが昨日あたりなわけですが、割りといい感じに作業が進んでいるというか、当時の動的コンテンツがない「ただのテキストサイト」みたいなのは、かなりサクッとできてしまうわけです。今の技術は本当にすごいですよね。

で、このホームページを書いていく過程で、細部にわたるレベルであまりにも「わかる人にはわかる」アレなホームページが作り上げられていって、「これはもう行けるだけ行くしかないでしょ」となってきました。最悪少なくともWeb黎明期（Windows95のちょい前ごろ？）の私のような「イマイチ輝けなかったホームページ書き」たちが見たら、もしかしたら喜ぶかも、みたいなモノは完成すると感じ始めました。最近のデジタルネイティブの若者たちといった「当時をわからない人には全くわからない」代物だという事も、はっきり理解できましたが(^q^)

言うてみたら、我々は当時を語り継ぐ古式ゆかしいホームページ作家だったのだと、思わずにはいられないわけであります。そのコンピュータ文化史的価値はあまりに大きいはずなのに、こういう当時の文脈というか、空気感はあまり注目を集めないわけです。まあテック業界的な文脈では当然と言えば当然なわけですが、そこの部分への個人的理解については置いておきます。

そんなわけでこのままだと、なにかに冷水を浴びせられて素に戻って全くやる気を失うまでは、永遠にテキストコンテンツを作り続けそうな感じなのですが、どこまで続くかは今のところわかりません。明日の朝には終わっているかもしれないです。それでもまあやってみます。

そんなわけでお楽しみくださいm(_ _)m
